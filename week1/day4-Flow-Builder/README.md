# Day 4 - Flow Builder Assignment
## What is Flow Builder?
Flow Builder is a Salesforce automation tool that helps users automate business processes without writing heavy code. It allows users to create workflows using screens, logic, conditions, and actions.
Companies use Flow Builder to:
- Reduce manual work
- Improve accuracy
- Save time
- Automate repetitive tasks

In this assignment, I created a Student Registration Automation Flow using Screen Flow and Decision elements.
# Types of Flows
## 1. Screen Flow
Screen Flow collects information from users using interactive screens.
Examples:
- Student registration form
- Feedback form
- Admission form
## 2. Record Triggered Flow
Record Triggered Flow runs automatically when records are created or updated.
Examples:
- Auto email after registration
- Update course seats automatically
- Notify faculty members

# Manual vs Automated Process
## Manual Process
In manual systems, staff members check forms one by one.
Problems:
- Slow process
- Human errors
- Duplicate records
- Difficult tracking
- More workload
## Automated Process
In automated systems, Salesforce checks conditions automatically.
Benefits:
- Faster processing
- Better accuracy
- Less manual work
- Consistent results
- Improved productivity

# Reflective Questions
## 1. Why do companies automate workflows?
Companies automate workflows to save time, reduce errors, and improve productivity.
## 2. What problems happen with manual processes?
Manual systems can cause delays, duplicate records, confusion, and human mistakes.
## 3. Difference between Screen Flow and Record Triggered Flow?
Screen Flow interacts with users using screens, while Record Triggered Flow runs automatically in the background.
## 4. Why is no-code automation powerful?
No-code automation helps users create workflows without advanced programming knowledge.
## 5. When should automation be avoided?
Automation should be avoided when human judgment or creativity is required.
## 6. How does automation improve consistency and productivity?
Automation follows the same process every time, reducing mistakes and increasing speed.
